This folder contains a implementation of a ExplicitEuler
solver in C that is interfaced using the python/C API. 
To make this folder executable you need to install the packages 
and compile the C scripts into a module. This is done by
using setuptools, by typing:

    python setup.py build

To run you excecute the python script:

    python main.py
